This is the game HiLo:

A number is generated and then a second number is picked.
If you guess whether or not the second number is higher or lower
then the first correctly, you get 100 points. If you guess wrong,
then you lose 75 points. The game continues untill
the player runs out of points or th player decides to quit.


technical rules:

Individual cards are represented as a number from 1 to 13.
The current card is displayed.
The player guesses if the next one will be higher or lower.
The the next card is displayed.
The player earns 100 points if they guessed correctly.
The player loses 75 points if they guessed incorrectly.
If a player reaches 0 points the game is over.
If a player has more than 0 points they decide if they want to keep playing.
If a player decides not to play again the game is over.
